yesday-sysutil project: System utilities for power users and DevOps engineers

Last updated on 28 October 2018

Posted by YesDay on 28 October 2018 Tags: system linux macos bash groovy ansible shell yesday-sysutil

The project yesday-sysutil provides a polyglot blueprint of system utilities for power users and DevOps engineers. It consists of Bash and Groovy shell scripts and Ansible playbooks, with each tool being selected as the best fit for specific use case. The project is configured with the Gradle build tool so that you can easily import it to your favourite IDE like IntelliJ and benefit from autocompletion and other IDE-related features. Also, skeleton shell scripts are provided to get you started with your own custom shell scripts as quickly as possible (see src/main/groovy/common/skeleton).
