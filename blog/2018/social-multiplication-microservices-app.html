Social Multiplication Proof of Concept (POC) application demonstrating a full Microservices Architecture

Last updated on 16 October 2018

Posted by YesDay on 16 October 2018 Tags: java microservices spring-boot spring-cloud-eureka ribbon zuul hystrix poc

Social Multiplication is a Proof of Concept (POC) application that demonstrates a full Microservices Architecture. The application embodies common Microservices patterns using Spring Boot, Spring Cloud Eureka, Ribbon, Zuul and Hystrix to implement Service Discovery, Load Balancing, the API Gateway pattern and a Circuit Breaker. Moreover, it implements end-to-end tests with Cucumber and an event-driven system with RabbitMQ.

Refer to the GitHub repository for further details and information on how to clone the source code and run the application along with the accompanying end-to-end tests.
